Ɛli Menas: amedyaz n At Leqser
Deg yiseggasen-a ineggura, aṭas n yimedyazen Iqbayliyen i d-yessuffuγen isefra-nsen d idlisen. Ayagi d ayen i d-irennun azal d tbaγurt i tutlayt d tsekla taqbaylit s umata. 

Γef  waya yewwi-d lḥal i wid mazal ur d-suffγen ara leqdicat-nsen, ad ɛeṛḍen ad d-afen tifrat amek ara d-yili usizṛeg n leqdicat-nsen. Ɣas akken qessiḥet tallit, γas yewɛer unnar.

Awal γef umedyaz:

Ʃli Menas, d yiwen seg waṭas n yimedyazen i d-tefka tmurt Taqbaylit. Ilul deg useggas di taddart n Bumnazel, tamnaḍt n At Leqṣer, agezdu n Tubiret, deg tmurt n Yiqbayliyen. 

D winn i yebdan isefra-ines imezwura mi yella d anelmad deg tesnawit. Ugar tamedyazt, mass Σli yura aṭas n tezlatin i ddeqs n yicennayen n Teqbaylit.

Ʃli Menas d winn yessefrayen γef waṭas n temsal d yisental, ama γef tayri, tameṭṭut, ansayen, tamurt, timetti d tudert n yall ass. Yerra tijmilin i yimeɣnasen d wid yessulin azal n Teqbaylit, d wayen nniḍen.
 
Rnu γer waya, yekki deg waṭas n temliliyin d tfaskiwin i d-yettilin γef tmedyazt, gar-asen-t: tafaska n At Leqser deg Tubiret, tafska n Itri Adelsan deg Uqvu, tafaska n Udrar n Fad deg Ayt Smaɛel, tafasa n Ayt Lqasem deg Buɣni, d waṭas nniḍen.

Deg wayyur n mayyu n useggas, amedyaz Ʃli Menas yessuffeγ-d amudd-is amenzu n tmedyazt γer tezṛigin "Terroir" n yiγrem n Tizi Wezzu. D adlis i wumi yefka azwel "Gar wallen d wallaɣ".  Ddan-d deg-s yisefra s Teqbaylit. Iga-as tazwert uselmad n tutlayt d yidles Amaziɣ mass Meqran Guzi.

Yeqqar-as deg usefru i wumi yefka azwel "Ajerbub":
"Tamart s aqerru tecrek
Tineslemt ziɣ deg ucebbub
Nɣan ṭṭbib mi yehlek
I leḥruz ldin lektub
Beddlen temlel s tebrek
D ṭṭlam i teffren leɛyub"
