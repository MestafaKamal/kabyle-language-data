Amaynut deg tezlit Taqbaylit: "Ayen a zheṛ-iw" n yilemẓi Ferḥat Muḥli 

Deg wayyur n fuṛar yezrin i d-yeffeɣ s wudem unṣib uḍebsi n ccna amezwaru n usdawan ilemẓi Ferḥat Muḥli. Aḍebsi-agi ddant-d deg-s sḍis n tezlatin. Yall tizlit d tedyant-is. 

D aḍebsi iwumi yefka yilemẓi Ferḥat, azwel "Ayen a ẓher-iw". Yeffeɣ-d ɣer teẓrigin "IDURAR”. Tineggura-agi yella ɣef uqerruy-nsent unaẓur n Teqbaylit Qasi Bussaɛd i yekkin zik deg ugraw i yettwasnen nezzeh "IDURAR" ɣer tama n Lwaliya Bussaɛd. 

Awal ɣef ucennay:

Ferḥat Muḥli ilul deg wass deg wayyur n tuber n useggas, deg Uweqqas, agezdu n Bgayet, di tmurt n Yiqbayliyen. Yezdeɣ deg taddart n Tbuɛlamin i yettafaṛen lɛerc n At buɛisi, di tɣiwant n Tizi n Berbar.

Taddart n Tbuɛlamin d tin i d-yefkan ayen din n yicennayen, gar-asen: Ṭaher Σissi i yerẓan asalu s wudem unṣib i tezlit deg taddart-a. Yessaweḍ yessekles kraḍ n tesfifin, syin akin yuɣal d aselmad n tutlayt d yidles amaziɣ. Akka tura a-t-an yettkemmil duktura-ines deg Fransa ɣef ccna amḥaddi deg umennuɣ ɣef tmagit. Syin iḍfer-it-id Ǧuder Ḥammu, Ǧuder Ḥmama, Lwennas Ǧerruǧ, Ǧuder Murad, Sifaks Yal, akka almi armi d ilemẓi Ferḥat Muḥli. 

Akka tura, ilemẓi Ferḥat, a-t-an yettkemmil leqṛaya-ines deg tesdawit n umeɣras Σebdereḥman U Mira, di bgayet.

Deg uḍebsi-agi amenzu, Ferḥat yecna sya ɣer da, kra n lfulkluṛ d kra n tezlatin i tmeɣriwin. Rnu ɣer waya, tella yiwet n tezlit s uɣanib n flaminku. Mi t-nesteqsa acimi, neɣ d acu I d sebba n tezlit-agi, yenna-d: “D tidet tella. D aɣanib i ḥemmleɣ nezzeh. Semḥaseɣ-as s waṭas. Ijbed-iyi, ur ẓriɣ ara amek. Sya ɣer zdat mi ara d-tban tafat, mi ara d-rnuɣ ad skelseɣ, flaminku ad d-yeddu. Ilaq.”

Ha-ten-an ihi yizwal n tezlatin i d-yeddan deg uḍebsi-agi : 
Ttxil-m anef-iyi, 
Yelli-s n Leqbayel, 
Ayen a zheṛ-iw, 
Tinn iruḥen, 
Ad am-d-iniɣ tidet, 
Gar tullas.
